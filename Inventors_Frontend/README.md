# Inventors(Im not really sure abt the name) - Emerging Tech Articles

Inventors is a React web app that displays the latest technology news and articles from multiple sources. Users can view articles, their sources, publication dates, and summaries. Links open in a new tab for full reading.

## Features

- Fetches live articles from the Inventors backend API.
- Displays article title, source, published date, and summary.
- Clickable links that open the full article in a new tab.
- Clean and responsive design for desktop and mobile.
- Built with React and styled with custom CSS.

## Getting Started


Tech Stack
React
Fetch API
CSS for styling
FastAPI backend <separate repository>
